
# Dynamic Input-Output Model for Climate Change Impact Assessment in Iran

## Purpose
This repository provides a **structural template** of a recursive dynamic input-output (IO) model with fuzzy optimization, developed as part of a scientific research project on climate change impact assessment in Iran. The code is intended to **illustrate the methodological framework** rather than serve as a standalone production tool. It is shared to support transparency, reproducibility, and scientific discussion in the peer-review process.

## Disclaimer
This code is a **simplified implementation** of the proposed methodology and is provided for **illustrative and academic purposes only**. It is not a fully operational model and does not include all data, parameters, or calibration steps required for real-world application. The primary goal is to clarify the dynamic IO logic and the optimization structure described in the corresponding manuscript.
